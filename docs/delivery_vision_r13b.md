## 一、西部绘制说明（全部在新投影坐标系直接落笔，未套东部仿射变换组）

填充 Skipper 预留的 5 个具名空图层（layer id 不变）：
- **layer-states-west**：秦（渭水流域·关中）`ellipse cx=150 cy=302 rx=118 ry=50 #C2A468`；
  楚（汉水—江淮）`ellipse cx=500 cy=590 rx=140 ry=82 #6F9E86`。色相自既有列国色系延展
  （秦=黄土金、楚=江南青绿），`fill-opacity=0.32` 与 layer-states 同。
- **layer-mountains-qinling**：秦岭——横亘渭水（北）与汉水（南）之间的东西向折线记号一列（7 记，x82→430，
  y≈375），风格同泰山/嵩山（stroke #8C8272、1.8、圆折）。
- **layer-rivers-wei**：渭水——自西东流经雍城一线（雍约169,311），入河水故道于潼关一带（约351,303，
  接东部河水西端点352.94,303）。
- **layer-rivers-han**：汉水——自西北汉中向东南经楚境，入江于汉口一带（约655,615）。
- **layer-rivers-jianghuai**：江（长江）—江淮下延——横贯南界自西向东、纳汉水、趋海（示南界方向感，非考据河道）。
- 水系皆 `stroke #7FA3A8 opacity 0.9`、非缩放线宽；渭 2.4 / 汉 2.2 / 江 3.0（江最巨，比照河水量级）。
- 走向校准（新投影参考点）：雍(169,311)、郢(508,634)、临淄(940,130)、曲阜(846,226)、新郑(616,319)
  逐一比对；渭经雍城一线、郢落汉水—江淮色块区，均吻合。
- **古名标注**：秦/楚（font 17，`data-state` 供海报放大）、渭水/秦岭/汉水/江水（font 12）并入既有
  `layer-labels` 两子组。
- **叠放次序修正**：Skipper 预留的 5 层原置于「古名标注」之后（会令色块盖住标注）；已整组上移至标注层
  之前，令标注压在色块与山水之上，与东部次序一致（layer id 不变，app.js/pipeline 按 id 取用、位置无关）。

## 二、首页海报化同步（app.js，本为西扩必修的对齐修复）

西扩仿射把东部色块搬走后，首页地图的**徽记簇与热区曾整体错位**（实测：徽记浮在色块左下方约百像素）。
本轮修复：
- `HOME_BADGE_POS`（齐/鲁/郑/晋 徽记簇心）按同一仿射 x'=0.7058824x+352.9412、y'=0.7222222y 换算到新投影。
- `buildHomeMap`：海报提档与热区生成一并纳入 `#layer-states-west`；新增 `ellipseFinalGeom()`——读每个
  色块**父组的合并变换**（东部有仿射矩阵、西部为单位阵），把 cx/cy/rx/ry 换算到最终 viewBox 坐标，
  再落于无变换的 hotLayer。故东西两套色块的热区与徽记皆与色块严丝合缝；秦/楚亦成键盘可达热区，
  无主角→「人物线整理中」。
