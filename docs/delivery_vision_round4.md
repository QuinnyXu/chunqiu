## 完成项

1. **亲至/相关区分**：轨迹只连「亲至」事件；地点锚点三态——亲至=主题色实心、仅相关=主题色空心环、无关=淡圈；图例常驻地图左下角。时间线角色签与弹窗事件列表同步标注「相关」。presence 空值按 conventions v1.2 视同亲至（防御性兜底，当前数据无空值）。
2. **平滑播放**：rAF 沿轨迹匀速插值，总时长 `min(8s, 1.2s×段数)`（下限 2s），到站自动播报并联动地点详情；同一按钮 播放→暂停→续播；切视图自动复位。逐站点击查看保留。
3. **免责声明**：地图右上角「示意底图 ⓘ」角标，点开说明（艺术示意非考据疆界；大势参照谭其骧通说；逐点依据见弹窗），aria-expanded 同步。
4. **排序统一**：时间线、刻度尺、地图轨迹全部改为 `(year_bce, sort_key, id)`，不再依赖 ID 顺序。实测鲁隐公线 -712 年 E032（秋，sort 10）正确排在 E002（冬，sort 20）之前——此处按 ID 排会错。
5. **资料库视图**（#view=library）：背景/考古/来源三标签 + 关键词过滤 + 列表-详情双栏；来源条目显示前缀含义（Z左传/S史记/G国语/P诗经/A考古/B现代研究）与 ctext 外链；顶部导航常驻入口 + 选人页底部入口；无需选人即可进入。
6. **八人预留**：选人栅格为 auto-fill 弹性布局，天然支持 8 卡；鲁庄公（苇墨绿 #55603A/鼓纹）、齐桓公（黄铜 #8A6D1F/连环三盟）主题色与徽记草案已入 design_notes 并生成 SVG；已实测两人数据未入库时显示"资料整理中"禁用占位卡。

## 验收偏差上报（重要）

**"E036 临淄应为空心点"与数据冲突**：文姜在临淄除 E036（相关）外，还有 E010（薨于齐之变，亲至）与 E012（孙于齐，亲至）两条亲至事件——地点聚合下临淄必为实心（亲至优先）。E036 的区分实际体现在：① 文姜轨迹不再因 E036 于前706 提前拐入临淄（现在临淄在 -694 才入轨迹，修复了此前的时序怪相）；② 时间线与弹窗中 E036 明确标「相关」。纯空心点可在齐襄公地图肉眼验证：讙、纪、郕三点均为空心。若领队坚持"E036 使临淄空心"，需改为事件级落点（同一地点画多个点），等裁定后再动。
