## 一、副题三候选（配合新站名「经纬春秋」）

三条各有取向、均可独立成立，非「一好两陪衬」。均 ≤10 字，短于旧副题（16 字）。旧副题「左传与国语中的人物、时间线与地图」把「国语」抬到与「左传」并列，且只点两书，已不合 round15 后「以《左传》为经，国语、三传、史记、诗经、诸子为纬/对照」的多源现实——三候选均以此为准绳修正。

### 候选 A · 「左传为经，诸书为纬」

- **取向**：方法论 / 多源分层。直解站名「经纬」之喻——《左传》独居「经」位为编年骨架，国语、三传（公羊、穀梁）、史记、诗经、诸子皆列「纬」位作参证对照，一句摆明本站的史料层级，与凡例「后出叙事分层」「T 层不单独立骨架」的口径同气。
- **理由**：八字工对，最"实"最准；「诸书」不写死书目、涵盖全部纬/对照来源，避免旧副题只点两书之弊。站名点「经纬」为题眼，副题恰好把题眼展开，题—注相生。
- **适用位置**：og 图、README、`<title>`/`og:title` 等**解释性场景通用且最佳**（这些场景欢迎一句把定位讲清）。作 logo 小字亦可，唯「经纬」二字与站名重复，近距离并置略有回声——若嫌重复，logo 处可让位给候选 B/C，og/README 用 A。

### 候选 B · 「牵系可游，处处有据」

- **取向**：可游走的体验（化用 Xiangtao「可游走的时空羁绊，处处有据」）。「羁绊」按站内简帛气质转译为「牵系」（人物间、人地间的牵连脉络，即关系网），「可游」承「可游走」（沿时间线、地图、关系网游走），「处处有据」承「处处有据」（条条挂来源编号）。关系＋行迹＋出处三事一句收束。
- **理由**：领队方向的正面落地；不用日系「羁绊」而取「牵系」，文白相济、不失牵连本义。最"活"，带邀请感与游走感，与结尾句「择一而入」、分享邀请语「处处有据」一脉。
- **适用位置**：logo 小字与 og 图**最宜**（钩子感强）；README 亦可。多源现实此条以「有据」间接承载（据在文献），显性程度低于 A/C——取向使然，非缺陷。

### 候选 C · 「系年缀地，众说相参」

- **取向**：编年地理 ＋ 异说对照的学术底色。「系年」＝时间线/编年，「缀地」＝活动地图（缀连地望），「众说相参」＝三传异同、史记与诗经诸子并存对照（对应凡例的分层与"异说并存则降级"精神）。
- **理由**：把本站两大主视图（时间线、地图）与"多源相参"的治学态度一句道尽；「众说相参」较旧副题的静态罗列更见方法，且不触碰任何可靠度分级用语。最"正"，定位清晰。
- **适用位置**：og 图、README **最宜**（学术定位一眼可辨）；`<title>` 亦佳；logo 小字可用，八字稍密但不挤。

> 三条取向速记：**A 讲来源层级（经/纬）· B 讲游走体验（牵系可游）· C 讲编年与对照（系年/众说）**。若须单选一条全站通用，A 与站名题注相生、最稳；若 logo 与 og 可分设，建议 logo 用 B、og/README 用 A 或 C。

## 三、头部双标小字格式定稿

### 现状

`site/index.html:32`：
```html
<h1 class="site-title">春秋人物志<small id="site-range">前773 — 前628</small></h1>
```
`#site-range` 的静态文本「前773 — 前628」**仅为 JS 未执行时的兜底**，实际由 `site/app.js:3394` 动态覆写：
```js
$("#site-range").textContent = yearLabel(m.year_range_bce.min) + " — " + yearLabel(m.year_range_bce.max);
```
现数据下运行时即渲染为「前773 — 前591」。**注意：静态兜底文本「前628」已过期**（round15 后上界为前591），应顺手校正。

### 定稿格式

**「双标」＝头部两枚并置标记**：品名标（站名「经纬春秋」）＋ 进度标（编年考订前沿小字）。进度标独立小字，不与品名混排成一句。

**进度标文案定稿：**

> **已考订至前591**

- **语义**：库中编年已整理至的**最晚年份**（考订前沿），随数据推进自动增大；表"时间覆盖到哪"，**非**"到此为止皆已考实"——与可靠度分级（high/medium/low）无关，不构成质量声明。用「考订」（整理编订及于）而非「考证无误」，与站内既有措辞「可考订的时间线」（`meta description`）同词同义，零冲突。
- **数字动态占位**：`591` 取 `meta.year_range_bce.max` 的绝对值经 `yearLabel()` 格式化（去负号、冠「前」字），与页脚「年代 前773—前591」同源（皆出 `site/data/meta.json`）。**HTML 内不写死**——静态文本改为占位「已考订至前591」或留空，由 `app.js` 注入 `#site-range` 实际值（现 `app.js:3394` 输出的是"起—讫"区间串，落地时须把该行格式改为 `"已考订至" + yearLabel(m.year_range_bce.max)`；此为 Vision/Skipper 施行项，本稿只定文案与取数口径）。

**备选（保留起讫双坐标，若须同时呈现起点）：**

> **前773 · 已考订至前591**

- 起点标 `773 = |min|`、前沿标 `591 = |max|`，中点用「 · 」分隔；两数同源 `meta.year_range_bce` 的 `min`/`max`。信息更全，但字数近乎旧区间串、"考订前沿"的推进感被起点稀释。**推荐主用「已考订至前591」单标**（承 Xiangtao 示例、进度感最强），起讫备选留作 og 或页脚需要完整跨度时取用。

- **理由**：旧「前773 — 前591」是静态区间读感，读者不知这是"全部即如此"还是"整理到这"；改「已考订至前591」把被动区间转为主动的"考订前沿"，暗示项目在持续推进（与 round15 刚推进至楚庄王卒年前591 的实况相符），且一字未碰可靠度术语。

## 四、关于页站名注脚拟文（顺手）

站名「经纬春秋」题眼为「经纬」，典出《左传·昭公二十八年》成鱄论德九目之末句「**经纬天地曰文**」（与「照临四方曰明」同段谥法之目）。据此拟关于页站名注脚一句，备用：

> **拟文**：站名「经纬春秋」，取《左传·昭公二十八年》「经纬天地曰文」之语——以《左传》为经，国语、三传、史记、诗经、诸子为纬，纵横以织春秋之文。

- **典源核对**：句见《左传·昭公二十八年》，为成鱄评魏献子举贤、申"文"德之谊时所举九目（「心能制义曰度……择善而从之曰比，经纬天地曰文」）之末。经传明文，reliability **high**。已网核该句确系昭公二十八年、确含「经纬天地曰文」与「照临四方曰明」二目同段（无法直连 ctext.org，改由多方检索复核一致）。
- **一句理由**：注脚把"经纬"从修辞落到治学——「经」是《左传》编年骨架、「纬」是众源参证，恰与副题候选 A、站内"多源分层"凡例同一口径；引经传本句而非《逸周书·谥法解》同文，既准且雅（该九目《逸周书》亦载，此处取《左传》本以合站内以传为宗的体例）。
- **适用位置**：关于页「这是什么」段或「参考文献」区末的一行小注均可；≤40 字，克制不喧。
