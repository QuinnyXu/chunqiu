## 一、播放聚焦改造（Xiangtao 反馈）

- **播放期间**：到站信息改为地图内叠加字幕条 `#play-caption`（`aria-live=polite`），格式"第N/M站 站名 · 前XXX年 · 事件标题（多事件以；相连）"，单行超长 ellipsis 截断；样式为半透明绢帛底 + 主题色左朱线，随播放逐站更新。不再打开下方卡片/抽屉、不滚动页面（旧行为是 `announceStation` 联动 `showPlace`，已移除）；播放中点击/回车站点一并抑制。
- **暂停或播完**：字幕条 250ms 淡出；此时点击站点恢复正常展开——内嵌=下方文档流卡片，全屏=底部抽屉。续播时字幕条带当前站文本复现。
- **内嵌与全屏统一**：字幕条节点随地图移入/移出全屏浮层（`openOverlay`/`closeOverlay` 中随 svg 迁移）。
- 实测（CDP 驱动 headless Edge，昭公 7 站线 + 文姜 8 站线）：播放中字幕可见有文本、地点卡未开、`scrollY=0`、点击站点被抑制；暂停后字幕淡出、点击恢复展开；播完字幕隐藏、按钮归位；全屏内字幕在浮层、不弹抽屉、退出归位内嵌。窄屏（400px 模拟）播放字幕正常，字幕条抬高至图例之上避让。

## 二、关系图"以人为中心"重构

- **默认 ego 视图**：从人物页/带人物语境进入关系页，以当前人物为中心。中轴纵向家系五档：祖辈 / 父母 / [同辈横排 ←] 本人 [→ 配偶并置] / 子女 / 孙辈；只保留出现的世代。家系分类依 conventions §6.5 方向约定（person_a 为 rel_label 主语）判定父母/子女、祖/孙；祖孙两代另沿"父母之父母、子女之子女"各推一代（连边须库中实有才绘出）。
- **两侧分组**：君臣/拥立/敌对/师友/其他 五类分列左右（按类型序贪心平衡两侧高度），组头可点击/回车折叠展开；同一人跨多类型（如周桓王之于郑庄公：君臣+敌对）只落一个节点、多边并入。家系已占位者不再入侧组，其非家系边（如襄公·私通·文姜）直接画到家系节点上。
- **边绘制**：8 类边色与图例沿用，medium/low 虚线；同排边走弧（家系向上、其余向下）、侧组边向外侧弯让开家系行、同对重边错开；触中心的边加粗提亮，其余淡化。`rel_label`/`source_note` 悬停 `<title>` 可见，点边或看右栏详情可读全文；侧组节点名后附小字 rel_label（>9 字截断）。
- **游走**：点击任意人物节点（或回车）以其为中心重绘；面包屑显示游走路径（超 5 级折叠为 …，可点回任一级），另有"← 返回上一个中心"按钮；侧栏同步该中心全部一度关系。主角节点主题色+徽记（中心放大+朱砂细环），配角小节点；主角可双击或经侧栏按钮跳时间线。
- **全景次级入口**：「全景 41 人」按钮切换到 round6 环形布局（原样保留未重排），新增「仅主角边」过滤（口径=**至少一端为主角**，88→66 边；无边可挂的配角淡化 0.25。若领队要更狠的"两端皆主角"口径=14 边，一行可改）；全景节点侧栏加「◎ 以其为中心」按钮切回 ego。首页「四十人关系图谱」入口明确直达全景。
- **680px**：窄屏首次进入默认折叠全部侧组（家系纵轴优先）；关系画布与地图同策略 `overflow-x:auto` + svg `min-width:640px` 保住可读尺寸。
- **自验基准实测通过**：文姜 ego = 上·齐僖公（父）；同辈·齐襄公、齐桓公；配偶·鲁桓公；下·鲁庄公（孙辈·子般，由庄公线自动推得）；侧·郑昭公（辞婚），襄公私通边落在同辈节点上。点齐桓公重绘后中轴变其家系（僖公/襄公/公子纠/文姜），面包屑"文姜›齐桓公"，返回键复原。另抽验齐桓公（17 边、五侧组）、郑庄公（三代五档全出）、鲁庄公（3 同辈+配偶哀姜）。

## 已知问题

1. 侧组节点旁的小字 rel_label 原样展示、不辨主语方向（"杀之"这类主宾敏感标签在压缩视图下可能歧义）；完整三元句式在悬停与侧栏均可见。如需彻底消歧需给 relations 表补反向措辞列，属数据层议题。
2. 全景工具条的"88 条关系"为库存总数描述，开过滤后不随之变为 66；下轮可改为动态计数。
3. 窄屏侧组"默认折叠"在每次页面加载时判定一次（matchMedia），会话中途拉伸窗口不重判。
4. ego 布局为规则网格而非避让求解器，个别稠密组合下边线仍可能擦过名签（已用弧线让行缓解，抽验八主角未见遮挡节点本体）。
5. 播放期间站点点击被完全抑制（含键盘），需先暂停再查看地点——此为任务口径的直接实现，若试用反馈想要"点击即暂停并展开"，一行可改。
6. 字幕条首站文本对读屏器可能少播一次（元素取消 hidden 与首次赋文同帧）；后续各站 aria-live 正常。
