# 姓名显示规则（xing/shi/ming/zi 拼装）

面向 Vision：`people` 表新增 `xing`（姓）、`shi`（氏）、`ming`（名）、`zi`（字）四列，供程序拼装"姓氏名字"一行说明。任一字段可能为空（空＝无考），拼装须**跳过空字段**。合入后本页由 Skipper 挪至 `docs/`。

## 一、核心区分（务必读）

- **先秦"姓"与"氏"不同**：姓别婚姻（女子系姓），氏别贵贱（男子称氏）。故拼装文案用"X姓Y氏"，不可写成"X氏Y姓"。
- **男子称氏不称姓，女子称姓**。数据里男子多有 `shi`、女子多只有 `xing`（姜、姬、嬴…"姜/姬/嬴"本身就是姓）。
- 空字段代表"无考"，不是"无"——**不要显示"（无）""null"**，直接省略该片段。留空原因见该人 `notes`。

## 二、拼装规则（按有无字段分档）

设 X=xing、S=shi、M=ming、Z=zi。推荐生成一行"名字说明"：

| 情形 | 判定 | 输出样式 | 例 |
|---|---|---|---|
| 姓＋名（诸侯/公室，无氏） | X 有、S 空、M 有 | `{X}姓，名{M}` | 鲁庄公 → 姬姓，名同 |
| 姓＋氏＋名（卿大夫） | X、S、M 皆有 | `{X}姓{S}氏，名{M}` | 管仲 → 姬姓管氏，名夷吾 |
| 姓＋氏＋名＋字（全备） | X、S、M、Z 皆有 | `{X}姓{S}氏，名{M}，字{Z}` | 管仲（全）→ 姬姓管氏，名夷吾，字仲 |
| 氏＋名（姓无考） | X 空、S 有、M 有 | `{S}氏，名{M}` | 鲍叔牙 → 鲍氏，名牙 |
| 姓＋名＋字（公室有字） | X、M、Z 有、S 空 | `{X}姓，名{M}，字{Z}` | 羽父 → 姬姓，名翚，字羽父 |
| 仅字可考 | 仅 Z 有 | `字{Z}` | （如某仅存字者）字羽父 |
| 仅姓（女子/名无考） | 仅 X 有 | `{X}姓（名无考）` | 文姜 → 姜姓（名无考） |
| 全空 | 皆空 | 不输出该行，或显示"姓名无考" | 颍考叔 → 姓名无考 |

## 三、通用拼装伪代码

```
seg = []
if X and S:      seg.append(f"{X}姓{S}氏")
elif X:          seg.append(f"{X}姓")
elif S:          seg.append(f"{S}氏")
line = "，".join(filter(None, [
    "".join(seg) if seg else "",
    (f"名{M}" if M else ""),
    (f"字{Z}" if Z else ""),
]))
# line 为空时显示 "姓名无考"；女子且 M 空可在"…姓"后加"（名无考）"
```

要点：
- `X姓S氏` 必须黏在一起（"姬姓管氏"），再与"名…""字…"以顿号"，"连接。
- `名`/`字` 段独立判空，各自缺则整段略去。
- 展示"名无考"文案仅用于女子或明确名缺者（M 空），可选；不要对每个空字段都加括注。

## 四、注意事项

- **女子**（文姜、武姜、哀姜、齐姜、骊姬、许穆夫人）：多只有 `xing`。"姜""姬"即姓，勿当名显示。骊姬之"姬"也是姓（骊戎姬姓），非名。
- **齐公室**统一 `姜姓吕氏`（齐襄公、齐桓公、齐僖公、公子纠、公孙无知、公子彭生）；隰朋为 `姜姓隰氏`。
- **异文/异说**：`竖刁` 的 `ming=貂`（从《左传》），显示名仍作"竖刁"——`name` 列是通行显示名，`ming` 是史料考订名，二者可不同，勿强行改 `name`。`易牙` 的 `ming=巫`（雍巫）同理。若要展示异文，读该人 `notes`。
- `alt_names` 与四列已核验一致，可继续用于检索/别名展示。
- 详尽依据、留空理由、可靠度（medium 标注）见 `CHANGES.md` 与各人 `notes`。
