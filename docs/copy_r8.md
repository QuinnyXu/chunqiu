## 一、关于页全部段落

### R1 · 定位（「这是什么」段）
- 现：春秋人物志是一个面向大众的春秋早期历史科普站：以文姜、鲁桓公、齐襄公、郑昭公、郑庄公、鲁隐公、鲁庄公、齐桓公、晋文公九条人物线为骨架，把《左传》《国语》等传世文献中的记事整理成可交互的时间线、活动地图与关系图谱。每一条事件都能回答"出自哪一年、哪部书"——数据表中逐条挂有来源编号与鲁君纪年。
- 新：以《左传》《国语》为经纬，缀连春秋人物的时间、行迹与关系。文姜、鲁桓公、齐襄公、郑昭公、郑庄公、鲁隐公、鲁庄公、齐桓公、晋文公，九条人物线织成可游走的时间线、地图与关系网。每一条都交代出处——哪一年、哪部书，数据表里都挂着来源编号与鲁君纪年。
- 理由：依领队示例定调，去"面向大众的科普站"产品腔；九人、出处、鲁君纪年等实质信息全保留，仅换以文言骨架的短句。

### R2 · 数据凡例·可靠度与亲至相关
- 现：**可靠度三级**：每条事件、地望与关系都标注 high / medium / low——high 可作骨架直接使用；medium 宜复核（多为追叙、纪年推算或异说并存）；low 只是线索。**亲至与相关**：人物与事件的关联区分"亲至"（史文明确记载其人身在事发地）与"相关"（与其相关但本人不在场，如遣使代行、为其议婚）；判定从严——史无明文即标相关，地图轨迹只连亲至事件。
- 新：**可靠度三级**：事件、地望、关系皆标 high / medium / low——high 可当骨架，medium 宜复核（多为追叙、纪年推算或异说并存），low 只作线索。**亲至与相关**：分"亲至"（史文明记其人身在事发地）与"相关"（事与其相关而本人不在场，如遣使代行、为其议婚）；判定从严——史无明文即标相关，地图轨迹只连亲至。
- 理由：术语与两项定义一字不动；仅删"每条…都""事件"等赘字、并列短句化。

### R3 · 数据凡例·坐标与后出叙事分层
- 现：**坐标确定性**：古地名的今地定位按"谭其骧《中国历史地图集》通说 → 杨伯峻《春秋左传注》 → 其他历史地理著作"的优先序考订，异说并存时坐标降级并注明依据；确址无考的给区域近似并标 low，个别事件明确"不落图"。**后出叙事分层**：《史记》等晚出文献中的戏剧性情节（管仲射钩、曹沫劫盟、病榻论三竖等）一律标注"后出叙事"层，与《春秋》经传的当代记事分开呈现，不混为一谈。
- 新：**坐标确定性**：古地今址按"谭其骧《中国历史地图集》通说 → 杨伯峻《春秋左传注》 → 其他历史地理著作"的次第考订，异说并存则坐标降级、注明所据；确址无考者给区域近似并标 low，个别事件径注"不落图"。**后出叙事分层**：《史记》等晚出文献里的戏剧情节（管仲射钩、曹沫劫盟、病榻论三竖），一律另标"后出叙事"层，与《春秋》经传的当代记事分列，不相混。
- 理由：地望优先序、low、"后出叙事"分层口径完整保留；"次第/所据/径注/分列"文言化并略缩。

### R4 · 数据凡例·地图示意
- 现：地图为示意图：列国底色与水系走向是艺术化简化，**不是考据疆界**。
- 新：地图只是示意：列国底色与水系皆艺术化简笔，**非考据疆界**。
- 理由："简笔／非"更凝练；"非考据疆界"的免责实质不变。

### R5 · 参考文献·首条引语
- 现：《春秋左传》《国语》《史记》《诗经》（原文均标注篇章，外链至 ctext.org）
- 新：《春秋左传》《国语》《史记》《诗经》（皆注篇章，外链 ctext.org）
- 理由：去"原文均…至"赘字；书名与外链保持准确。
- 附：其余书目（杨伯峻《春秋左传注》、谭其骧《中国历史地图集》、韩茂莉等历史地理著作及各故城考古）为专名，**保留不动**。

### R6 · 版本与反馈
- 现：发现史实错误或地望异议？欢迎到 GitHub Issues 反馈，史实纠错请附出处；数据与代码见 开源仓库。
- 新：读到可疑处？欢迎经 GitHub Issues 见告，纠错请附出处；数据与代码俱在 开源仓库。
- 理由："见告／俱在"文白相济，去"反馈"官腔；含义不变。
- 附：`about-version`「当前版本 v0.6（…）」及版本更迭列表为事实性元数据，**保留不动**。

### R8 · 文姜线
- 现：文姜线 —— 最戏剧性的命运线。齐国公室之女、鲁国国母，前694年那场变故之后，她的每次出行都被史官记下一笔。
- 新：文姜线 —— 齐国公室之女、鲁国国母；前694年那场变故后，她每一次出行都被史官记下一笔。
- 理由：并作一句，去"最戏剧性的命运线"评断腔；"齐僖公之女、鲁桓公夫人、鲁庄公之母"以"公室之女／国母"概括，前694（桓公十八年鲁桓死于齐）史实不动（high）。

### R9 · 郑庄公线
- 现：郑庄公线 —— 《左传》开篇的雄主。克段于鄢、周郑交质、繻葛之战，都在这条线上。
- 新：郑庄公线 —— 《左传》开篇的雄主：克段于鄢、周郑交质、繻葛之战，皆在此线。
- 理由：冒号连缀为一句，"皆在此线"替"都在这条线上"，简净；三事（隐元年、隐三年、桓五年）不动。

### R10 · 齐桓公线
- 现：齐桓公线 —— 从内乱到霸业。自莒返国、管仲为相、北杏与柯之盟，一路走到九合诸侯。
- 新：齐桓公线 —— 从内乱到霸业：自莒返国、管仲为相、北杏之会与柯之盟，终至九合诸侯。
- 理由：一句化；"北杏之会与柯之盟"较原"北杏与柯之盟"更准（庄十三年北杏为会、柯为盟），且不触碰柯之盟中曹沫劫盟的后出叙事层。

### N1 · 晋文公线（新增第四条）
- 现：无（本轮新增）
- 新：晋文公线 —— 流亡十九年的归国者：自骊姬之乱出奔，遍历列国，暮年返晋即位，终成城濮之霸。
- 理由：领队指定的第四条入门线；"流亡十九年"用其定调（前655—636 之传统计数），以"暮年"避开生年推算（medium），骊姬之乱出奔、城濮成霸皆经传明文（high）。
