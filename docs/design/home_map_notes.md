# 首页地图化 · 设计小记（布局与交互定稿）

作者：Vision，r12。提案：Xiangtao（领队裁定采纳）。适用：site/ 首页（#/）。
上位文档：docs/design/design_notes.md（v1.4 起收录本页要点），docs/conventions.md v1.6。

## 1. 立意

首页从「名单」变成「舆图」：一眼见列国大势，点国入人。底图复用
`assets/map/base_map.svg` 的列国色块图层，做**海报化**处理——色块饱和度提一档
（fill-opacity 0.32 → 0.5）、国名字号放大（17 → 26，颜色加深），水系与山地保留为
底纹层不再交互。人物地图（#/p/X/map）的底图参数不变，两处共用同一份 SVG 文件。

底图本轮新增**晋**色块与国名（此前底图无晋，而晋文公线已上线）：晋核心区
（曲沃—绛—翼）按投影公式核算落于 x≈147–172、y≈297–308，色块取 cx 172, cy 298；
并补一条汾水（北来南折，于 x≈75, y≈372 处入河），走向合真实地理大势。色块仍是
势力示意、非考据疆界，图上免责口径不变。

## 2. 布局

- **桌面（>680px）地图模式**：左为底图（占满余宽），右为「列国人物」面板
  （320px，同地图页 side panel 规格）。点国 → 面板列出该国人物卡；点卡 → 时间线。
  分组卡片列表（第 7 轮视图）在地图模式下隐藏。
- **手机（≤680px）地图模式**：底图在上（容器横向滚动，SVG min-width 640px，
  与人物地图同策略），分组卡片列表在下同屏纵排；右侧面板隐藏。点有主角之国 →
  平滑滚动定位到下方该国分组；点无主角之国 → 底图下状态行显示一句提示。
  小屏不依赖精细点击：热区加大（见 §3），且列表始终在场兜底。
- **列表模式**：标题行「列表模式 / 地图模式」切换按钮，切回第 7 轮完整分组视图
  （国别选项卡 + 分组卡片）。模式记忆走 URL hash 参数：`#/`＝地图（默认）、
  `#/?home=list`＝列表；**不使用 localStorage/sessionStorage**（平台约束）。
  旧链接 `#/`、`#person=…` 行为不变。

## 3. 交互与热区

- 每国一个 SVG 热区组（`role="button"`、`tabindex="0"`、Enter/空格触发）：
  命中椭圆＝色块椭圆外扩 18 底图单位，覆盖国名与徽记簇；hover/focus 显示暖赭
  虚线环，选中态常显。键盘 Tab 顺序即底图声明顺序。
- **主角徽记簇**：有主角的国（齐 4、郑 4、鲁 3、晋 1）在国名旁排一行小徽记
  （r=13 主题色圆底＋白线徽记，间距 31），簇心坐标为每国一处布局锚点
  （HOME_BADGE_POS，属美术布局配置，不是史料落点——史料地点仍一律走投影公式）。
  徽记簇整体在热区之内，点徽记即点国。
- **无主角之国**（周、卫、宋、陈、蔡、纪、许）：点出一句提示——
  `国名 · 气质注——人物线整理中`。气质注为界面文案，配置于 app.js
  `STATE_EPITHET`（本轮补齐七国），保持探索感、不做死链。
- 面板人物卡与列表卡**同一构件**（personCardLi），全站统一尺寸（见 §4）。

## 4. 人物卡统一尺寸（Xiangtao 反馈 1 根治）

卡为四行定式：徽记（56px 定宽）＋姓名行＋生卒行＋单行简介。规则：

- `min-height: 8.5rem`，简介、姓名行、生卒行一律单行截断（ellipsis）；
- 信息列 `min-width: 0`（flex 截断生效的前提）；
- 缺项（如武姜无生卒）不改变卡高，只留行空。

首页网格、首页地图面板、任何后续复用处皆同此规格。

## 5. 人物子导航升格（Xiangtao 反馈 3）

- 人物名：主题色、衬线加粗，字号升一档（0.85 → 1.08rem），徽记 20 → 24px；
- 未激活视图标签：淡墨 `#7A7166` 加深至正文玄墨 `#2E2A24`；
- 激活态不变（主题色深底白字）；
- 整条 bar 上下内边距加大，与下方内容留白拉开（子导航在场时 main 顶部留白 +0.5rem）。
- 桌面与手机同一套规则。

## 6. 无障碍

- 国家热区 Tab 可达、Enter/空格触发，aria-label 为「某国：N 位人物」或
  「某国：人物线整理中」；
- 模式切换按钮 `aria-pressed` 标状态；面板 `aria-live="polite"`；
- 不禁用页面级缩放；地图容器横滚不劫持纵向滚动。

## 7. 明确不做

- 首页地图不做拖移/缩放交互（那是人物地图的事），保持「海报」性格；
- 不在首页地图落任何城邑点——城邑属人物地图语境；
- 不做国界描边（色块即示意，描边会伪装成考据疆界）。
